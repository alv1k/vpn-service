# Скрипты для тестирования API

## Описание

Эти скрипты позволяют протестировать работу API AmneziaWG для создания и управления VPN-клиентами.

## Скрипты

- `test_amneziawg_api.ps1` - PowerShell скрипт для Windows
- `amneziawg_curl_commands.sh` - Bash скрипт с curl командами
- `test_amneziawg_api.bat` - Batch скрипт для Windows
- `test_amneziawg_api_simple.sh` - Упрощённый bash скрипт без зависимости от python
- `simple_test_api.sh` - Простой bash скрипт для проверки API

## Использование

См. подробное описание в `../docs/API_TESTING.md`
